# Vue.js + vue-router + webpack 技术栈搭建的仿 CNODE 社区

### 项目模块组件：
- Header 模块
- PostList 模块
- Article 模块
- Slider 侧边栏模块
- UserInfo 用户个人中心模块
- Pagination 分页组件的开发

### 采用了如下技术栈：
- vue.js 计算属性
- vue.js 的内置指令和事件的绑定
- vue.js 的自定义事件和触发
- vue-router 路由的跳转和监听
- 父子组件之间的数据传递
